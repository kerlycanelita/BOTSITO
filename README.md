# BOTSITO

A lightweight Discord bot built with **Node.js** and **discord.js** designed to integrate Minecraft server utilities and manage PvP tier information directly from Discord.

---

# Features

- Discord **Slash Commands**
- PvP **Tier VS management commands**
- Minecraft server utility integration
- Fast **guild-specific command deployment**
- Built with **discord.js v14**

---

# Commands

| Command | Description |
|------|-------------|
| `/tiernow` | Displays the current status of the PvP tier system |
| `/setup` | Configures the Tier VS system |

---

# Tech Stack

- **Node.js**
- **discord.js v14**
- **minecraft-server-util**
